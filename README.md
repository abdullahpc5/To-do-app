# Simple Demo To-Do App

This is a simple to-do list application that allows users to add tasks, mark them as completed, and organize their daily activities. The app is created using HTML, CSS, and JavaScript.

## Features

- **Add Tasks:** Users can input their tasks into the text field and add them to the list by clicking the "Enter" button.
- **Mark as Completed:** Users can mark tasks as completed by selecting the radio button next to each task. Completed tasks will be crossed out.
- **Responsive Design:** The app is styled to be responsive and user-friendly.

## Project Structure

The project consists of the following files:

1. **index.html:** The main HTML file that structures the content of the to-do app.
    - Contains the input field, button, and to-do list container.
    - Integrates external styles from `style.css` and functionality from `script.js`.

2. **style.css:** The CSS file responsible for styling the to-do app.
    - Styles the overall layout, text, input field, and buttons.
    - Includes two color themes (black and blue) for different elements.

3. **script.js:** The JavaScript file that handles the interactive functionality of the to-do app.
    - Adds tasks to the list when the "Enter" button is clicked.
    - Allows users to mark tasks as completed by clicking the radio button.
